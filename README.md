### Running in dev
1. Clone this repository
2. Execute
```bash
npm install
```
3. Install Nest CLI
```bash
npm i -g @nestjs/cli
```
4. Up the database
```bash
docker-compose up -d
```

## Dependencies
* NestJS
* MongoDB
